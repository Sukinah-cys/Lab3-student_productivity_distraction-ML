This project performs exploratory data analysis (EDA) on a dataset that examines how academic habits, lifestyle factors, and digital distractions affect student productivity.

## Objective
To analyze student productivity patterns and identify key factors influencing productivity scores through statistical summaries and data visualization.

## Dataset
### Student Productivity & Distraction Dataset
The dataset contains 20,000 student records describing academic habits, lifestyle behaviors, and productivity levels.

## Key Features
* Study hours per day
* Sleep duration
* Phone usage
* Social media usage
* Gaming hours
* Stress level
* Focus score
* Attendance percentage
* Final grade
* Productivity score (target variable)

## Dataset Characteristics
* Total Records: 20,000
* Total Features: 18
* Missing Values: None

## Tools & Technologies
* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Analysis Performed
### Data Quality Checks
* Checked for missing values
* Identified rows containing missing data
* Checked duplicate records
* Examined dataset dimensions
* Inspected data types

### Statistical Analysis
* Generated descriptive statistics
* Summarized dataset features

### Data Visualization
* Histogram of productivity scores
* Bar plot of productivity by gender
* Box plot for detecting outliers
